# Student Pass/Fail Predictor

A simple machine learning project that predicts whether a student will **pass or fail** based on their grades, study time, absences, and past failures.  
Uses **Logistic Regression** with scikit-learn and provides interactive input for new students.

## Features
- Train/test split evaluation
- Logistic Regression model
- Interactive input for new student predictions
- Displays pass/fail probability
